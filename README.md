# System-Update-Modifier
Utility tool to detect pending system updates and display actionable notifications like “System Updates Available: 5. Run 'sudo apt upgrade' to install.”
#  System Update Modifier

A lightweight Bash script that helps you check, list, and install system updates on APT-based Linux systems.

##  Features
- Check for available updates  
- List all upgradable packages  
- Install updates automatically  
- Simple menu-based interface  
- Requires sudo for safe execution  

##  Requirements
- Ubuntu/Debian-based system  
- Bash shell  
- Internet connection  
- Sudo privileges  

##  Setup
```bash
chmod +x modifier.sh
```
